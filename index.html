Introduction

This is Natalia Kusek's coursework - Data Visualisation with JavaScript.

The submission contains of:
  • css/ directory containg bulma.io CSS
  • js/ directory containing JavaScript files
    • index.js is the main script. It loads all other functions.
    • data.js contains readings CSV (original Boonsong Lekagul waterways readings.csv has been included directly into JavaScript code as modern browsers doesn't allow to read local files) and waterways map (Waterways Final.jpg) converted using QGIS to TopoJSON format
    • visualisations.js contains specifications of Vege-Lite charts
    • functions.js helper functions
    • filtering.js functions to generate filtering controls for Overview tab
    • options.js functions to generate options controls for Overview tab
    • selection.js functions to generate selection controls for Details and Comparison tabs
    • navigation.js functions to generate navigation bar
    • lib/
      • moment.js library to manipulate dates
      • Font Awesome library providing fonts for bulma.io
      • Papa Parse library to parse CSV files
      • Vega-Lite library to generate charts/visualisations

Question 1

Characterize the past and most recent situation with respect to chemical contamination in the Boonsong Lekagul waterways.

Finding: Descreasing values

Lead and Chromium have a descreasing value trend since 1998 until 2016 for Boonsri, Busarkhan, Chai, Kannika, Kohsoom, Sakda, Somchair.
Each location have similar trends which can be divided into 3 periods
  • 1998 - 2003, 2003 - 2008 and 2008 - 2016 for Lead
  • 1998 - 2003, 2003 - 2013 and 2013 - 2016 for Chromium
Kohsoom and Somchair have the highest readings in the first period for both measurements and stay at the top in other two periods.
The second period have the highest number of outliers.

Specification

Used marks:
  • line representing trend for each location over time for given measurement (line chart)
  • line representing aggregated values of measurement for each location (boxplot chart)
Used channels:
  • horizontal position (x) on 1st and 5th plots to show how the value changes over time
  • veritical position (y) on 1st and 5th plots to show the value of the measurement
  • color to distinguish between locations
  • vertical position (y) on 2nd-4th and 6th-8th plots to show each location separately
  • horizontal position (x) on 2nd-4th and 6th-8th plots to show values
Used features:
  • vconcat to make a dashboard
  • transform/filter field measure having values Lead and Chromium
  • transform/filter field location to included locations of interest
  • transfrom/filter field date to make 3 ranges for boxplot charts
  • resolve/scale shared to have the same scale for all 3 boxplot charts
The data has been filtered to include the locations that had data since 1998 only. It's to exlude locations that were introduced later and we can't reason about the trend the same way as for locations included in the chart.
The line chart show the general trend of the measurements over years for each location which was chosen to show one quatitative attribute (value of measurement) and one ordered key attribute (time).
The 3 boxplot charts shows changes in min, max, Q1, Q3 and median values for the selected periods and locations. This shows exactly that the values are lower in each period. It also shows that some of the locations have higher values than the others.
Toolips are added to be able to identify outliers.
We used boxplot to get summary statistics for the distribution of a quantitative attribute which allowed us to characterise distribution, find aoutliers, extremes and averages where each original atrribute showed five derived values.
Please note tooltips breaks when selection is introduced for concatented charts, so I've decided to leave tooltips instead of selection for the that reason.

Finding: sudden drop and rise

Chlorodinine and Methylosmoline have a sudden change in readings.
Chlorodinine readings dropped in all locations, while Methylosmoline rised in Kohsoom, Chai and Somchair and dropped for the rest.
Chlorodinine and Methylosmoline are recorded since 2014.

Finding: Data sampling not consistent

Sampling locations are not distributed evenly across all 4 waterways.
  • Waterway 1 (blue) has a single location: Decha
  • Waterway 2 (red) has a single location: Tansanee
  • Waterway 3 (orange) has 3 locations: Achara, Somchair, Sakda
  • Waterway 4 (turquoise) has 5 locations: Boonsri, Chai, Kannika, Kohsoom, Busarakhan
Locations: Achara, Decha, Tansanee have been introduced in 2009.
Boonsri, Chai have the most number of readings recorded. Kannika, Sakda are placed after them. All other locations (inlcuding the onces introduced in 2009) have similar number of readings.
The above means Waterway 4 (turquoise) has best coverage, while Waterway 1 (blue) and Waterway 2 (red) have the worst.
It's also worth mentioning Waterway 1 (blue), Waterway 2 (red), Waterway 4 (turquoise) don't have reading locations at the bottom of the map, so not the whole area is covered.

Samplings are not evenly distributed across the time.
There's a peak in number of readings between 2005 and 2008 and summer 2008 and 2010.

There's an high increase in Water temperature readings in Chai in 2016, which looks to be taken daily since then.

Not all measurements are taken across all locations.

Measurments are not taken with similar time periods.

The first chart shows general trend over years and allows to select a date range to investigate using other 4 charts. This chart was used to find trends and part-to-whole relationships between one quatitative attribute (values of measurement) and two categorical key attributes (time and location).
